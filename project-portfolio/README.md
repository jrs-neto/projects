## Portfólio Demonstração – Front End & UX/UI Design

Este é um **projeto de demonstração** de portfólio pessoal, criado para mostrar habilidades em Front End e UX/UI Design usando **HTML** e **CSS** .

[Clique aqui]() para acessar o Website.

## Sobre o Projeto

O portfólio demonstra:

-**Introdução**: Estrutura de apresentação com título e área para foto de perfil.

-**Experiência**: Seção de projetos fictícios para demonstrar layout de experiência profissional.

-**Formação**: Exemplos de formação acadêmica e cursos intensivos.

-**Contato**: Área de contato simulada para mostrar estrutura de layout.

O design é **moderno**, **responsivo** e **organizado**, utilizando **CSS Grid** e **Flexbox**, ideal para estudos e demonstração de habilidades.

## Tecnologias Utilizadas

-**HTML5**

-**CSS3** (modularizado por seções)

## Principais Características

**Design responsivo** usando CSS Grid e Flexbox, adaptando-se a diferentes dispositivos.

**Seções organizadas** para Experiência, Formação e Contato, com layout limpo e moderno.

**Tipografia** e **cores consistentes**, garantindo boa legibilidade e identidade visual.

**Estrutura modular de CSS**, facilitando manutenção e personalização.

**Interatividade planejada**, com espaço para adição de scripts em JavaScript.

**Layout escalável**, pronto para incluir novos projetos, cursos ou informações de contato.
